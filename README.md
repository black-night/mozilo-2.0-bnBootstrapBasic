mozilo-2.0-bnBootstrapBasic
===================
Layout für moziloCMS 2.0 (https://github.com/mozilo/mozilo2.0)
Ein einfaches Layout mit Navigation oben auf der Seite und einer festen Breite. Das Layout passt sich automatisch an kleinere Displays an. Es sind verschiedene Farbthemen enthalten.

Es wird das Plugin Bootstrap benötigt . (https://github.com/black-night/mozilo-2.0-Bootstrap)
